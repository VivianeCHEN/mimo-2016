Outils à installer

  1. Installer l'IDE PyCharm CE
  2. Outils de versionnement Git
  3. Créer ou utiliser un compte Github
  4. Suivre le tutoriel Github ici
  5. Installer Docker
  6. Executer dans une ligne de commande:
    docker pull julienroussel/mimo-2016

Bibliothèques Python utiles

  • notebook (Projet Jupyter): conda install notebook
  • lxml (site): conda install lxml
  • CherryPy (site): conda install CherryPy
  • beautifulsoup4 (site): conda install beautifulsoup4
  • Jinja2 (site): conda install Jinja2
  • SQLAlchemy (site): conda install SQLAlchemy
